# AItutor - 高校专属智能助教与学情评估系统

AItutor 是一款基于大语言模型与 RAG（检索增强生成）技术深度定制的高校智能化教学辅助平台。系统旨在通过 AI 力量协助教师进行精准学情分析，并为学生提供沉浸式的课后辅导与自我测评环境。

---

## 项目介绍

本项目针对高校教学场景中的“资料繁杂、反馈滞后、缺乏针对性练习”等痛点，构建了集资料分析、智能测验、知识库答疑于一体的数字化教学闭环。

系统核心优势：
1. 语义化资料分析：深度解析学生上传的讲义、笔记、论文，构建多维度的学力画像。
2. 引导式学情测评：AI 扮演苏格拉底式助教，通过启发式提问引导学生自主思考，而非直接给出答案。
3. 动态知识库管理：支持课程资料的自动分块与向量化，确保 AI 答疑内容的学术严谨性与溯源性。

## 系统架构

系统采用模块化单体架构，通过 Spring AI 深度集成大语言模型，并利用 Redis Stream 实现高耗时任务的异步化处理。

任务处理流程：
1. 资料上传请求：保存原始文件并提取元数据。
2. 异步任务下发：通过 Redis Stream 发送解析与向量化指令。
3. 消费端处理：Consumer 消费消息，执行长文本拆分、Embedding 嵌入及向量入库。
4. 状态反馈：前端通过 WebSocket 或轮询获取任务处理进度（待处理/处理中/已完成/失败）。

## 核心功能

### 学情资料分析模块
1. 多格式深度解析：支持 PDF, DOCX, DOC, TXT 等学术常用格式。
2. 异步分析流水线：基于 Redis Stream 确保长文档解析的系统稳定性。
3. 学力维度评估：从知识广度、理解深度、逻辑严密性等五个维度生成分析结果。
4. 学习计划导出：支持将 AI 生成的分析结论导出为正式的 PDF 学业评估报告。

### 课后智能测评模块
1. 启发式出题逻辑：基于学生上传资料，智能生成覆盖核心知识点的测评题目。
2. 引导式问答交互：AI 会根据学生的回答进行实时追问，模拟真实助教辅导场景。
3. 分批次评估机制：解决长文本评估中的 Token 限制问题，确保评估结果的连贯性。
4. 错题集与建议：自动归纳测评中的薄弱环节，并给出针对性的复习建议。

### 课程知识库模块
1. 文档向量化存储：支持课程讲义、参考书目的自动切片与向量化入库。
2. RAG 增强答疑：集成向量检索，确保 AI 回答具备文档支撑，减少模型幻觉。
3. SSE 流式响应：采用 Server-Sent Events 技术实现流畅的打字机式对话体验。

## 快速部署

### 环境要求
1. JDK 21 或更高版本。
2. Node.js 18 或更高版本。
3. PostgreSQL 14+ (需安装 pgvector 扩展)。
4. Redis 6+。

### 部署步骤
1. 配置环境变量：设置 AI_BAILIAN_API_KEY。
2. 修改 application.yml：配置数据库、Redis 及 S3 存储连接信息。
3. 后端启动：执行 ./gradlew bootRun。
4. 前端启动：进入 frontend 目录，执行 pnpm install 与 pnpm dev。
